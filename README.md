# Todoey

Todoey is a simple to-do list app built using SwiftUI. It allows users to add, edit, complete, and delete reminders in an easy-to-use interface.

## Features

- Add new reminders  
- Edit existing reminders  
- Mark reminders as completed or incomplete  
- Swipe to delete reminders  
- Smooth UI with Dark Mode support
